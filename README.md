# Sensys

A minimal, robust Android app for device and sensor information, featuring a clean Material 3 UI/UX and modern branding.

## Features
- Displays device and supported sensor information
- Visualizes only supported sensors (hides unsupported)
- Clean, minimal Material 3 design
- No display of sensor version or power
- Downloadable APK included at the repo root
- Custom app icon and branding

## Getting Started
1. **Download the APK**: Go to the [Releases](https://github.com/arunkg005/Syspulse/releases) section to download the current version APK.
2. **Install on your Android device** (you may need to allow installation from unknown sources)

## Building from Source
1. Clone the repository:
   ```sh
   git clone https://github.com/arunkg005/Syspulse.git
   ```
2. Open in Android Studio or build with Gradle:
   ```sh
   ./gradlew assembleDebug
   ```
3. The APK will be in `app/build/outputs/apk/debug/`
